# Spreaker API SDK - Javascript


### Supported Browsers

 * IE8+
 * Chrome
 * Firefox
 * Safari
 * Opera


### Contribute

#### Setup

Install `gulp`, `serve` and local node modules:

```bash
npm install -g gulp
npm install -g serve
npm install
```

Run `serve`:

```bash
serve .
```

Open your browser at [http://localhost:3000/example/auth.html](http://localhost:3000/example/auth.html)
